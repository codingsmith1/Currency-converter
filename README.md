# 💱 Currency Converter

A real-time currency converter web app with a dark background and a clean, minimal card UI featuring flag icons for each currency.

---

## 📸 Preview

> A light purple card centered on a dark background, showing an amount input, From/To currency dropdowns with flag emojis, a swap button, and a "Get Exchange Rate" button.

---

## ✨ Features

- 💵 Enter any numeric amount to convert
- 🌍 Select **source currency** (e.g., USD 🇺🇸)
- 🌏 Select **target currency** (e.g., INR 🇮🇳)
- 🔄 **Swap** button to instantly flip the currencies
- 📈 Fetches and displays the **live exchange rate**
- Shows result inline: e.g., `1 USD = 93.79576452 INR`

---

## 🛠️ Technologies Used

| Technology | Usage |
|------------|-------|
| **HTML5** | Form inputs, dropdowns, and page structure |
| **CSS3** | Dark background, card styling, flag icon layout |
| **JavaScript** | Exchange rate API calls, swap logic, DOM updates |

---

## 🌐 API Used

> Exchange rate data is fetched from a free currency API (e.g., [ExchangeRate-API](https://www.exchangerate-api.com/) or similar).

---

## 📱 Responsive Design

Tested on:
- S — 320px
- M — 375px
- L — 435px
- Tablet — 768px

---

## 🚀 Getting Started

1. Open `index.html` in any modern browser
2. Enter the amount you want to convert
3. Select **From** and **To** currencies
4. Click **Get Exchange Rate**

> Requires an internet connection to fetch live rates.
